# RAG Engine

O RAG da MindLoop foi projetado para:

- alta precisão  
- contextualização normativa  
- explicabilidade  
- tolerância a ruído  

---

## Pipeline

1. Normalização  
2. Chunking inteligente  
3. Indexação FAISS / Milvus  
4. Reranking  
5. Geração guiada por contexto
