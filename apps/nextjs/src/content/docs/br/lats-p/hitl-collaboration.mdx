# Human-in-the-Loop (HITL)

HITL é uma parte essencial da filosofia IAA™.

O objetivo não é substituir humanos,  
mas **amplificar** sua capacidade de decidir.

---

# Quando o agente aciona HITL?

1. Quando a entropia é alta  
2. Quando há respostas conflitantes  
3. Quando o contexto é insuficiente  
4. Quando a lógica normativa é ambígua  
5. Quando o usuário explicitamente solicita revisão  
6. Quando o modelo demonstra incoerência  

---

# Como funciona?

```txt
Agente → "Tenho dúvida neste critério"
        → Mostra caminhos possíveis
        → Mostra justificativas
Usuário → Seleciona o caminho correto
Agente → Salva na memória

---

O que o HITL ensina ao agente?

- qual interpretação é preferida
- como lidar com ambiguidades
- diferentes estilos de decisão da empresa
- variações entre ativos
- exceções normativas
