# LATS-P

O **LATS-P** (Language Agent Tree Search – Probabilistic) é o framework de raciocínio estruturado da MindLoop.  
Ele combina:

- árvores de decisão dinâmicas  
- interpretação semântica via LLM  
- análise probabilística  
- backtracking inteligente  
- detecção de incerteza por entropia  
- colaboração humano-máquina (HITL)

É o núcleo cognitivo dos Agentes Assistidos (IAA™).

---

## O que o LATS-P resolve?

- Ambiguidade em textos reais  
- Tomada de decisão normativa  
- Explicabilidade e auditabilidade  
- Redução de erros de interpretação  
- Consistência entre diferentes operadores
